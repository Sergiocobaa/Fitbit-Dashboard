# Fitbit Dashboard

Dashboard personal de salud con datos de Fitbit Air via Google Health API v4.

## Setup

```bash
npm install
cp .env.local.example .env.local
# Rellena las variables en .env.local
npm run dev
```

## Variables de entorno necesarias

```
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GOOGLE_REFRESH_TOKEN=
NEXTAUTH_SECRET=
```
